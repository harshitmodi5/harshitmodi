About TicTacToe

  • TicTacToe is an iOS and android native application.
  • This game is a digital version of the arcade game which we used to play on a paper.
  • This application will let you play unlimited games and save game data.
  • User can store game data and check later who won the game.
  • User can re-play the whole game and track back to the steps played before.
